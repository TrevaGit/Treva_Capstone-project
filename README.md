# To-Do List App

This is the Capstone Phase 1 project for ALX Front-End ReactJS.  
It is a simple To-Do List App where users can add, view, edit, delete, and mark tasks as completed or pending.

## Features

- Add tasks
- View tasks
- Mark tasks as completed/pending
- Edit tasks
- Delete tasks
- Optional: Persist tasks using localStorage

## Tech Stack

- React
- Tailwind CSS
